color는 css에서 중요한데, 색상을 우리가 원하는걸 만들어서 사용해야하기 때문임!
기본적으로 제공하는 yellow이런 컬러를 사용하기보다 만들어서 원하는 색을 사용함
(xlsxwriter를 할때 16진수로 사용했던거 처럼?)

그리고 변수도 굉장히 중요한데, 많이 쓰임, 말그대로 값을 저장할 수 있게해주고
css를 프로그래밍 언어처럼 보이게 해줌~

컬러 시스템
1. 16진수
#000000 #FFFFFF 처럼 

2. RGB / RGBA
252, 206, 0

RGBA (Alpha)
252, 206, 0, 0.5
가장 뒤에 붙는 A는 투명도를 의미하는데, 백그라운드 컬러로 사용될때 투명도를 조정할 수 있음!


컬러를 여러곳에 사용하기 위해서는 복사 붙여넣기를 반복적으로 해야하는데,
이는 굉장히 쿨하지 못함!

그래서 :root라고 불리는 element에 변수를 추가할 수 있음
먼저 CSS에서 변수는 --두번 그리고 변수이름, 공백은 무조건 -로 채워야 한다
--main-color / --name-of-human 이런식으로.

그리고 변수를 사용하는 방법은 var(변수이름) 이런식으로 사용하면 된다.
background-color: var(--main-color)

이를 css custom properties 라고함!
